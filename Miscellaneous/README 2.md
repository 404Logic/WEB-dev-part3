##  Project Overview

**Spares & Brothers** is a modern, responsive website for a family-owned automotive business based in Vleifontein, Limpopo, South Africa. The business specializes in selling quality used vehicles and genuine spare parts.

The project began as a **basic HTML prototype** and has since evolved into a **fully styled HTML + CSS website** with a cleaner design and better user experience.
