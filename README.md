# Spares & Brothers Website

A responsive website for Spares & Brothers, a local business in Limpopo specializing in quality used vehicles and genuine spare parts.

## Features

- **Responsive Design**: Optimized for all devices (desktop, tablet, mobile)
- **Multi-page Website**: Home, About, Products, Testimonials, and Contact pages
- **Interactive Elements**: Product filtering, contact forms, and smooth animations
- **Modern UI**: Clean design with Bootstrap framework and custom CSS
- **SEO Optimized**: Proper meta tags and semantic HTML structure

## Pages

1. **Home (index.html)**: Hero section, featured products, services overview, and call-to-action
2. **About (Pages/about.html)**: Company story, team introduction, mission/vision, and timeline
3. **Products (Pages/products.html)**: Product catalog with filtering and search functionality
4. **Testimonials (Pages/testimonials.html)**: Customer reviews and statistics
5. **Contact (Pages/contact.html)**: Contact information, form, and map integration

## Technologies Used

- **HTML5**: Semantic markup and structure
- **CSS3**: Custom styling with Bootstrap framework
- **JavaScript**: Interactive functionality and form handling
- **Bootstrap 5**: Responsive grid system and components
- **Font Awesome**: Icons and visual elements
- **Google Fonts**: Typography (Montserrat and Roboto)
